# plasma.sh

Source for the [plasma.sh](https://plasma.sh) landing page — the home of **Plasma**, the open-source real-time intelligence platform.

This repository is served via GitHub Pages at the apex domain `plasma.sh` (see `CNAME`).

## What's here

- `index.html` — the single-page landing (self-contained: inline CSS + JS, no build step)
- `CNAME` — custom domain (`plasma.sh`)
- `.nojekyll` — serve files as-is, bypassing Jekyll

## Related

- Toolchain: [`plasmactl`](https://github.com/plasmash/plasmactl) and its plugins
- Docs: [docs.plasma.sh](https://docs.plasma.sh)
- Install: `curl -sSL https://get.plasma.sh | sh`
- Community: [Discussions](https://github.com/orgs/plasmash/discussions)
